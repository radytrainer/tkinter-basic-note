## Tkinter basic note

#### Import Tkinter library
```python
import tkinter as tk
```

#### Create basic tkinter Window object
```python
import tkinter as tk

window = tk.Tk()

# Your code here
window.mainloop()
```

#### Create title on window
```python
import tkinter as tk

window = tk.Tk()

# create title on window
window.title("My Game Title")

# Your code more here

window.mainloop()
```
